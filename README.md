# XPAnalyzer Web Portal

Este es el portal web oficial del proyecto **XPAnalyzer**, que incluye:

- Página principal (`index.html`)
- Página de inicio de sesión (`login.html`)
- Página de registro (`register.html`)
- Archivos CSS y recursos adicionales

El objetivo del proyecto es crear un sitio simple y funcional que sirva como:
- Portal de descargas
- Sistema básico de autenticación (visual)
- Interfaz limpia para pruebas en Windows Sandbox
